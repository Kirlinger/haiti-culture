# 🇭🇹 Haïti — Découvrir la Perle des Antilles

A static HTML website celebrating Haitian culture, history, spirituality, and tourism.

## 📁 Project Structure

```
haiti-culture/
├── index.html                  # Home page
├── about.html                  # About page
├── history.html                # History of Haiti
├── vodou.html                  # Haitian Vodou
├── lwa.html                    # The Lwa (spirits)
├── tourism.html                # Tourism & destinations
├── contact.html                # Contact page
├── culture-vivante.html        # Living culture
├── creativite-populaire.html   # Popular creativity
├── diaspora-engagee.html       # Engaged diaspora
├── jeunesse-potentiel.html     # Youth & potential
├── patrimoine-national.html    # National heritage
├── memoire-universelle.html    # Universal memory
├── capacite-adaptation.html    # Adaptation capacity
├── fragilite-economique.html   # Economic fragility
├── inegalites-sociales.html    # Social inequalities
├── instabilite-politique.html  # Political instability
├── pressions-securitaires.html # Security pressures
├── reconstruction-durable.html # Sustainable reconstruction
├── services-essentiels.html    # Essential services
├── css/
│   └── style.css               # Main stylesheet
├── js/
│   └── script.js               # Main JavaScript file
└── images/                     # SVG illustrations
```

## 🚀 Getting a Copy of the Code

### Option 1 — Clone with Git

```bash
git clone https://github.com/Kirlinger/haiti-culture.git
cd haiti-culture
```

### Option 2 — Download as ZIP

1. Go to the repository page: <https://github.com/Kirlinger/haiti-culture>
2. Click the green **Code** button near the top right.
3. Select **Download ZIP**.
4. Extract the ZIP file to a folder on your computer.

### Option 3 — View individual files

You can browse and copy any individual file directly on GitHub by clicking its name in the file list above.

## 🖥️ Running Locally

No build step is required. Simply open `index.html` in your browser:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows (PowerShell)
start index.html
```

Or use a local development server:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```
